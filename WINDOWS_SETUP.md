# Windows Installation Guide

## Quick Start (Automated Installation)

### Prerequisites
- Windows 10 or Windows 11
- Internet connection
- Administrator access

### One-Click Installation

1. **Download or clone this repository**

2. **Right-click `install.bat` and select "Run as Administrator"**

3. **Press any key when prompted to start installation**

The installation script will automatically:
- ‚úÖ Install Chocolatey package manager
- ‚úÖ Install Python 3.11
- ‚úÖ Install Node.js 18
- ‚úÖ Install MySQL 8.0
- ‚úÖ Create database and user
- ‚úÖ Install all dependencies
- ‚úÖ Run database migrations
- ‚úÖ Create admin user account

**Installation takes approximately 10-15 minutes** depending on your internet speed.

### Running the Application

After installation completes, simply run:

```batch
run.bat
```

This will:
- Start the backend server (Port 8000)
- Start the frontend server (Port 5173)
- Automatically open your browser to http://localhost:5173

## Advanced Options

### Custom MySQL Password

If you want to set custom passwords during installation:

```powershell
.\setup.ps1 -MySQLRootPassword "YourRootPass" -AppDBPassword "YourAppPass"
```

### Skip MySQL Installation

If MySQL is already installed on your system:

```powershell
.\setup.ps1 -SkipMySQLInstall
```

## Manual Installation (If Automated Fails)

### 1. Install Prerequisites Manually

Download and install:
- [Python 3.11+](https://www.python.org/downloads/)
- [Node.js 18+](https://nodejs.org/)
- [MySQL 8.0+](https://dev.mysql.com/downloads/installer/)

### 2. Setup Database

Open MySQL Command Line and run:

```sql
CREATE DATABASE inventory_management;
CREATE USER 'inventory_user'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON inventory_management.* TO 'inventory_user'@'localhost';
FLUSH PRIVILEGES;
```

### 3. Backend Setup

```batch
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

REM Create .env file manually (see configuration section above)

alembic upgrade head
```

### 4. Frontend Setup

```batch
cd frontend
npm install

REM Create .env file manually
```

### 5. Start Servers

**Terminal 1 (Backend):**
```batch
cd backend
venv\Scripts\activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

**Terminal 2 (Frontend):**
```batch
cd frontend
npm run dev
```

## Troubleshooting

### Installation Failed - "Chocolatey not found"

**Solution:** Manually install Chocolatey first:
1. Open PowerShell as Administrator
2. Run:
```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force
[System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072
iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
```
3. Restart PowerShell and run `install.bat` again

### MySQL Connection Error

**Solution 1:** Check MySQL is running:
```batch
net start MySQL
```

**Solution 2:** Verify credentials in `backend/.env`:
```env
DATABASE_URL=mysql+pymysql://inventory_user:AppPass@123@localhost:3306/inventory_management
```

### Port Already in Use

**Backend (8000):**
```batch
cd backend
venv\Scripts\activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8001
```

**Frontend:** Vite will automatically use the next available port (5174, 5175, etc.)

### Permission Denied Errors

**Solution:** Ensure you're running as Administrator:
- Right-click Command Prompt or PowerShell
- Select "Run as Administrator"

### Firewall Warnings

Windows Defender may prompt for network access:
- ‚úÖ Allow access for Python
- ‚úÖ Allow access for Node.js
- ‚úÖ Allow access for MySQL

### Python/Node Not Found After Installation

**Solution:** Restart your terminal or computer to refresh environment variables.

## Uninstalling

To completely remove:

1. **Uninstall via Chocolatey:**
```batch
choco uninstall python nodejs mysql -y
```

2. **Remove application folder**

3. **Drop database (optional):**
```sql
DROP DATABASE inventory_management;
DROP USER 'inventory_user'@'localhost';
```
